# Modular Telegram Assistant Bot 🤖

A conversational Telegram bot built with Python, designed to respond to user messages with static and dynamic replies using natural language patterns.

## ✨ Features
- /start, /help, /about commands
- Static responses (greetings, farewells, compliments)
- Dynamic responses (math solving, joke telling)
- Regex pattern matching for flexible input
- Group & private chat handling
- Modular response engine
- Secure token management using '.env'
- Hosted on replit

## 🛠️ Built With
- Python
- [python-telegram-bot]
- 'pyjokes', 'simpleeval', 'dotenv'

## 🚀 Run It Locally
1. Clone this repo
2. Create a `.env` file with:
TOKEN=your_bot_token
BOT_USERNAME=@your_bot_username
3. Run: 'python main.py'
